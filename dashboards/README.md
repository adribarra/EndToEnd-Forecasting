# Dashboards

This folder contains visualization apps for forecast outputs and model performance.

## Current app

- `streamlit_app.py`: Streamlit dashboard for:
  - champion model display
  - MAE/RMSE/MAPE comparison
  - metric bar chart from latest training report

## Run locally

```bash
streamlit run dashboards/streamlit_app.py
```

## Data source

The dashboard reads artifacts generated by batch training:

- `artifacts/metrics/training_report_*.json`

Run `forecast-train` before launching the dashboard.
